STM32H5 PB Counter                                                          

Home page LED control PB status bar

This page allows you to continuously get the user push button counter value. The user push button is connected to the
NUCLEO-H563ZI board. The value is updated, automatically, each 1s with the last push button input value.
You could increment it by pressing the user push button and checking that the counter value is updated.
