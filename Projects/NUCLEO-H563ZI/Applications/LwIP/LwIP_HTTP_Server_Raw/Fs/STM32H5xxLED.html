Home page LED control PB status bar


This page allows you to control the three LEDs: LED1, LED2, LED3 located in the NUCLEO-H563ZI board. To set an LED
on/off you have to check/uncheck its corresponding checkbox. Then you have to click on "Send" button to submit the new LEDs
status. Finally check the NUCLEO-H563ZI-board to see if the desired LEDs are on/off.
